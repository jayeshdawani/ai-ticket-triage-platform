# Feature Flag Experimentation Platform

A backend-focused feature flag and experimentation platform for controlling feature rollout behavior.

## Problem
Teams need a safe way to turn product features on/off and gradually roll them out to users.

## Target Users
- Software teams
- Student developers
- Small SaaS projects

## Core Features
- Create feature flags
- Enable/disable flags
- Evaluate flags for users
- Percentage rollout
- Simple experiment tracking

## Tech Stack
- Python
- FastAPI
- PostgreSQL
- React
- TypeScript

## Current Status
Week 1 setup in progress.

## Roadmap
- Week 1: Repo and concept setup
- Week 2: Flag CRUD and evaluation endpoint
- Week 3: Percentage rollout logic
